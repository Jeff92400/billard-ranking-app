# Billiard Icon Setup

## To use your custom billiard icon:

1. Save your billiard ball icon image as: **`billiard-icon.png`**
2. Place it in this directory: `/frontend/images/`
3. The icon should be square (e.g., 64x64px or 128x128px)
4. PNG format with transparent background recommended

All HTML pages will automatically use this icon once saved.
